# My_first_project

This is my first Python project created in Jupyter Notebook.  
It is an interactive program that takes personal data from the user and displays it with data types and memory details.

## Niche
Beginner Python Project / User Input & Data Handling

## Features
- Takes user input (name, age, height, favourite number)
- Converts data into appropriate types (int, float, string)
- Displays data type and memory ID
- Calculates approximate birth year

## Tools Used
- Python
- Jupyter Notebook

## Output Example
- Displays entered data
- Shows data types (str, int, float)
- Calculates birth year
